## Project Overview

This project is a Geometric Brownian Motion (GBM) Simulator. It visualizes multiple simulated paths of a GBM process, which is commonly used in financial mathematics to model stock prices and other assets. The animation shows the evolution of asset prices over time under stochastic dynamics.

## How to Fork

1. Click the "Fork" button at the top right of the repository page on GitHub.
2. Clone your forked repository to your local machine:
   ```bash
   git clone https://github.com/YOUR-USERNAME/Geometric-Brownian-Motion-Simulator.git
   ```
3. Install the required dependencies (see below).
4. Run the simulation using:
   ```bash
   python main.py
   ```
